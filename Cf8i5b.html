문제

시작요일과 말일을 입력받아 달력을 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
일(0),월(1),화(2),수(3),목(4),금(5),토(6) 중에서 
이번달 1일은 무슨요일인가요 : 2
이번달 말일은 언제인가요 : 9
   일   월   화   수   목   금   토
              1    2    3    4    5
    6    7    8    9

코드 빈칸 채우기

#include <stdio.h> 
 
int main() { 
	const int SUNDAY = 0, SATURDAY = 6;		 
	int firstDay = 0, lastDay = 0, dayOfWeek = 0; 
 
	printf( "일(0),월(1),화(2),수(3),목(4),금(5),토(6) 중에서 \n이번달 1일은 무슨요일인가요 : " ); 
	scanf( "%d", &firstDay ); 
	printf( "이번달 말일은 언제인가요 : " ); 
	scanf( "%d", &lastDay ); 
	printf( "   일   월   화   수   목   금   토\n" );	 
 
	( dayOfWeek = SUNDAY; 										 
	                        dayOfWeek < firstDay;  
	                                             dayOfWeek ) { 
		printf( "     " ); 
	} 
	 
	 ( int day = 1; 									 
	                  day  lastDay;  						 
	                                 day, dayOfWeek ) { 
		printf( "", day );	 
		if ( ( dayOfWeek % 7 ) == SATURDAY ) { 
			dayOfWeek = SUNDAY - 1; 
			printf( "\n" ); 
		} 
	} 
 
	return 0; 
}
